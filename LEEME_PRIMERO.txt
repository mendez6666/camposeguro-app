CAMPOSEGURO v1.8 - PRUEBA FIRMS EN RENDER

Nuevo:
- Página Prueba FIRMS.
- Permite probar Santa Cruz, Bolivia o Sudamérica.
- Muestra URL enmascarada, status, conteo de filas y primeras líneas de respuesta.
- Sirve para saber si el problema es llave, BBOX, fuente o conexión desde Render.

En Render:
Build Command:
pip install -r requirements.txt

Start Command:
uvicorn app:app --host 0.0.0.0 --port $PORT

Variables:
FIRMS_MAP_KEY=tu_llave_real
FIRMS_AREA_BBOX=-64.9,-20.6,-57.0,-13.0
FIRMS_DAY_RANGE=1
FIRMS_SOURCES=MODIS_NRT,VIIRS_SNPP_NRT,VIIRS_NOAA20_NRT,VIIRS_NOAA21_NRT
EMAIL_ENABLED=false

Después de desplegar:
1. Abrir /prueba-firms
2. Probar Santa Cruz 5 días
3. Probar Bolivia 5 días
4. Si da filas, volver a Inicio y Actualizar monitoreo.CAMPOSEGURO v2.0 - ALERTA TEMPRANA 24H

La consulta técnica puede descargar hasta 5 días para evitar que Render quede en 0.
Pero las alertas se generan únicamente con focos de las últimas 24 horas.

Variables recomendadas en Render:
FIRMS_QUERY_DAYS=5
ALERT_WINDOW_HOURS=24
FIRMS_DAY_RANGE=1
